# Histopathology CV

Two tasks on the Computer Vision Mini Project (PUMA) dataset.

**Task 1** — Tissue segmentation: 3-class pixel-wise classification (**Tumor / Stroma / Other**).  
**Task 2** — Nucleus classification: 3-class patch classification (**Tumour / Lymphocyte / Histiocyte**).

## Results Summary

### Task 1 — Tissue Segmentation

| Model | Avg Dice | Tumour | Stroma | Other |
|-------|----------|--------|--------|-------|
| UNet exp5 (Focal + Lovász) | 0.463 | 0.840 | 0.404 | 0.147 |
| **UNet exp5 + CRF** | **0.502** | 0.841 | 0.512 | 0.154 |
| AE exp2 (MSE pretrain + full finetune) | 0.406 | 0.829 | 0.336 | 0.051 |
| Baseline (provided) | 0.467 | — | — | — |

### Task 2 — Nucleus Classification

| Configuration | Params | Acc | Macro F1 |
|--------------|--------|-----|----------|
| **2a Exp 4: NucleiResNet + ECA + Mixup** | ~2.8M | **0.759** | **0.754** |
| 2a Exp 5: ResNet-18 (ImageNet) | ~11.2M | 0.745 | 0.737 |
| 2a Exp 2: NucleiResNet base | ~2.8M | 0.742 | 0.732 |

## Repo Structure

```
histopathology-cv/
├── 1_shared/              # Shared data pipeline and losses (Task 1)
│   ├── data_processing.py # TissueDataset, GeoJSON→mask, HED augmentation
│   └── losses.py          # FocalLoss, LovászSoftmax, DiceLoss, build_criterion
│
├── 1a_unet/               # Task 1a — UNet end-to-end segmentation (6 experiments)
├── 1b_autoencoder/        # Task 1b — Autoencoder pre-training + SegDecoder (4 experiments)
│
├── 2_data/
│   └── extract_patches.py # Nucleus patch extraction (train / validation / contrastive)
│
├── 2a_classifier/         # Task 2a — End-to-end nuclei classification (5 experiments)
└── 2b_contrastive/        # Task 2b — SimCLR pre-training + frozen head (4 experiments)
```

## Data

Expected at `../Coumputer_Vision_Mini_Project_Data/` relative to this repo:

```
Coumputer_Vision_Mini_Project_Data/
├── Dataset_Splits/
│   ├── train/         (image/ + tissue/ + nuclei/)
│   ├── validation/    (image/ + tissue/ + nuclei/)
│   └── test/          (image/ + tissue/)
├── task2_patches/     (generated by extract_patches.py)
│   ├── train/
│   ├── validation/
│   └── contrastive/   (leakage-safe set for SimCLR pre-training)
└── Task2_Test_Set/    (provided .npy patches)
```

## Setup

```bash
pip install -r requirements.txt
```

## Quick Start

```bash
# Task 1a — best UNet experiment:
python 1a_unet/run.py --exp 5
python 1a_unet/run.py --exp 6   # CRF post-processing on exp5

# Task 1b — best autoencoder experiment:
python 1b_autoencoder/run.py --exp 2

# Task 2 — extract patches first (run once):
python 2_data/extract_patches.py

# Task 2a — best classifier:
python 2a_classifier/run.py --exp 4

# Task 2b — best contrastive experiment:
python 2b_contrastive/run.py --exp 3
```
